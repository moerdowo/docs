## Authorization

  Example:

  `Authorization | Bearer Paste-Your-API-Key-Here`
</ResponseField>

## Path Parameters

<ParamField path="productId" type="string" required>
  The unique identifier of the product associated.
  
  Example:

  `f5103e7c-9e30-467b-bc32-a96fdcb65849`
</ParamField>

<ParamField path="memberId" type="string" required>
  The unique identifier of the member from product membership.

  Example:

  `PUYSW40N`
</ParamField>

## Response

Successful Response

### Main Structure (Root Object)

<ParamField path="statusCode" type="integer" required>
  Indicates the status of the activation process.
</ParamField>

<ParamField path="message" type="string" required>
  Additional messages related to the activation process.
</ParamField>

<ParamField path="customerEmail" type="string" required>
  Email address of the customer.
</ParamField>

<ParamField path="customerName" type="string" required>
  Full name of the customer.
</ParamField>

<ParamField path="customerMobile" type="string" required>
  Mobile phone number of the customer.
</ParamField>

<ParamField path="status" type="string" required>
  Current status of the customer (e.g., "active")
</ParamField>

<ParamField path="nextPayment" type="string" required>
  ISO 8601 timestamp (UTC) of the customer's next payment.
</ParamField>

<ParamField path="expiredAt" type="string" required>
  ISO 8601 timestamp (UTC) of the customer's membership expired.
</ParamField>

<ParamField path="memberId" type="string" required>
  Membership code associated with the customer.
</ParamField>

<ParamField path="membershipTierId" type="string" required>
  ID of the customer's membership tier.
</ParamField>

<ParamField path="membershipTier" type="object">
  Detail of the customer's membership tier.
</ParamField>

<ParamField path="id" type="string" required>
  ID of the membership tier.
</ParamField>

<ParamField path="name" type="string" required>
  Name of the membership tier.
</ParamField>
