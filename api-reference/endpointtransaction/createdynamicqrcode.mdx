---
title: "POST Create Dynamic QRCode"
description: "Endpoint used to create dynamic QRCode"
---

<RequestExample>

```bash Request Example
curl --location 'https://api.mayar.id/hl/v1/qrcode/create' \
--header 'Authorization: Bearer Paste-Your-API-Key-Here' \
--data-raw '{
   		"amount": 10000
	}
'
```

</RequestExample>

<ResponseExample>

```json Response Example 200
{
    "statusCode": 200,
    "messages": "Success",
    "data": {
        "url": "https://media.mayar.id/images/resized/480/a30ed45f-976b-490f-b97c-72c90d1e8d9d.png",
        "amount": 10000
    }
}
```

</ResponseExample>
