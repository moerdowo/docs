---
title: "Get Re-open Single Payment Request"
openapi: "GET /endpointqrpayment/reopen"
description: "Endpoint used to change single payment request status from closed to open"
---

<RequestExample>

```bash Request Example
curl --request GET 'https://api.mayar.id/hl/v1/payment/close/e890d24a-cfc0-4915-83d2-3166b9ffba9e' \
--header 'Authorization: Bearer Paste-Your-API-Key-Here'
```

</RequestExample>

<ResponseExample>

```json Response example if success
{
    "statusCode": 200,
    "messages": "success"
}
```


```json Response example if failed
{
    "statusCode": 200,
    "messages": "failed"
}
```

</ResponseExample>

Endpoint:

<CodeGroup>

```Production Production
https://api.mayar.id/hl/v1/payment/open/(productId)
```


```Sandbox Sandbox
https://api.mayar.club/hl/v1/payment/open/(productId)
```

</CodeGroup>

### Headers

<ResponseField name="Authorization" type="string" required>
  Example:

  `Authorization | Bearer Paste-Your-API-Key-Here`
</ResponseField>

### Param

<ParamField path="productId" type="string" required="false">
  A unique identifier obtained when creating an single request payment. If accessed through the dashboard, it can be found at the URL on the request payment details page.

  Example:

  `e890d24a-cfc0-4915-83d2-3166b9ffba9e`
</ParamField>
