Endpoint:

<CodeGroup>

```Production Production
https://api.mayar.id/hl/v1/invoice/open/(productId)
```


### Headers

<ResponseField name="Authorization" type="string" required>
  Example:

  `Authorization | Bearer Paste-Your-API-Key-Here`
</ResponseField>

### Param

<ParamField path="productId" type="string" required="false">
  A unique identifier obtained when creating an invoice. If accessed through the dashboard, it can be found at the URL on the invoice details page.

  Example:

  `f774034d-d9cc-43a0-97d8-a2520c127f03`
</ParamField>
