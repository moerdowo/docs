### Overview

The Checkout Setting feature on Mayar allows sellers to edit Checkout Page base on Requirements or Idea by Seller, it helping Seller to customizing template also with copywriting. With this feature, Seller can implement their creative idea for create their own template Checkout Page System.

### How to Create a Checkout Setting?

Follow these simple steps to create a Checkout Setting in Mayar:

- Chose one of product on list.

  ![Screenshot2025 08 13124715 Pn](/images/Screenshot2025-08-13124715.png)
- Select "View" from the product hyperlink.

  ![Screenshot2025 08 13124845 Pn](/images/Screenshot2025-08-13124845.png)
- Select Checkout Setting menu on List at right side menu bar.

  ![Screenshot2025 08 18131601 Pn](/images/Screenshot2025-08-18131601.png)
- You will see the "Checkout Setting" form. Fill in the required details, Including:

  | Wording Payment Button     | Field for change copywriting of Payment Button.           |
  | -------------------------- | --------------------------------------------------------- |
  | T&C Setup dropdown         | Terms & Conditions if needed.                             |
  | Wording Discount Button    | Field for change copywriting of Discount Button.          |
  | Hide Code Coupon           | Hide or unhide Code Coupon input field.                   |
  | View Popup Transaction     | Alert message for showing PopUp Transcation from Costumer |
  | Notes under payment button | For give disclaimer before Checkout                       |
