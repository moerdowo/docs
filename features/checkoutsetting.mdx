### Overview

The Checkout Setting feature on Mayar allows sellers to edit Checkout Page base on Requirements. With this feature, you can set wording for the checkout button, add terms and condition, add final note, etc.

### How to access the Checkout Settings?

Follow these simple steps to access the Checkout Setting in Mayar:

- Chose one of product on list.

  <img
    src="/images/Screenshot2025-08-13124715.png"
    alt="Screenshot2025 08 13124715 Pn"
    style={{ width:"39%" }}
  />
- Select "View" from the product hyperlink.

  ![Screenshot2025 08 13124845 Pn](/images/Screenshot2025-08-13124845.png)
- Select Checkout Setting menu on List at right side menu bar.

  <img
    src="/images/Screenshot2025-08-18131601.png"
    alt="Screenshot2025 08 18131601 Pn"
    style={{ width:"57%" }}
  />
- You will see the "Checkout Setting" form. Fill in the required details, Including:

  | Option                     | Description                                               |
  | -------------------------- | --------------------------------------------------------- |
  | Checkout Button Wording    | Change the checkout button wording                        |
  | T&C Setup dropdown         | Terms & Conditions if needed.                             |
  | Wording Discount Button    | Field for change copywriting of Discount Button.          |
  | Hide Code Coupon           | Hide or unhide Code Coupon input field.                   |
  | View Popup Transaction     | Alert message for showing PopUp Transcation from customer |
  | Notes under payment button | Final note and isclaimer before Checkout                  |
