### Overview

The Checkout Setting feature on Mayar allows sellers to edit Checkout Page base on Requirements or Idea by Seller, it helping Seller to customizing template also with copywriting. With this feature, Seller can implement their creative idea for create their own template Checkout Page System.

### How to Create a Checkout Setting?

Follow these simple steps to create a Checkout Setting in Mayar:

1. Chose one of product on list.

   ![Screenshot2025 08 13124715 Pn](/images/Screenshot2025-08-13124715.png)
2. Select "View" from the product hyperlink.

   ![Screenshot2025 08 13124845 Pn](/images/Screenshot2025-08-13124845.png)
3. Select Checkout Setting menu on List at right side menu bar.

   ![Screenshot2025 08 18131601 Pn](/images/Screenshot2025-08-18131601.png)
4. You will see the "Checkout Setting" form. Fill in the required details, Including: Wording Payment Button: Field for change copywriting of Payment Button. T&C Setup dropdown: if no then there's not terms & condition, if yes give link for T&C checkout. Wording Discount Button: Field for change copywriting of Discount Button . Hide Code Coupon: hide or unhide Code Coupon that avail for it. View Popup Transaction: alert message for showing PopUp Transcation from Costumer.
