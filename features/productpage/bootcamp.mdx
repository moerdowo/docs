## Overview

Mayar Kelas Cohort / Bootcamp is a online-course class that enables you to create a structural class & course within various study program more ease to access. Kelas Cohort / Bootcamp can be created using the Mayar Web Apps. This documentation focuses on creating and customizing payment links through the Mayar Dashboard.

## Creating a Kelas Cohort / Bootcamp

### Navigate to Kelas Cohort / Bootcamp

- Log in to your Mayar Dashboard
- From the Sidebar menu, click on Kelas Cohort / Bootcamp

### Create Kelas Cohort / Bootcamp

- Click on the Kelas Cohort / Bootcamp Link button
- Fill in the necessary details for the Kelas Cohort / Bootcamp, with Date Month Year of Class that you have been set before.
- Payment Type: You can Provide free class & paid class
- Upload Image Cover: The image is a placeholder for uploading a cover image, likely for a promotional purpose.
- Upload Video Cover: The Video is a placeholder for uploading a Video, likely for a promotional purpose.
- Note: Leave the amount field empty for Optional field.
- You can choose to create the Kelas Cohort / Bootcamp in any Price that you have setup.
- Click on Buat Kelas Cohort / Bootcamp.

### Share Kelas Cohort / Bootcamp

- Once the Kelas Cohort / Bootcamp is generated, you can copy/paste or share it directly with your customer.
- When the customer clicks the Kelas Cohort / Bootcamp, available payment methods linked to your Mayar account will be displayed.

### Editing Kelas Cohort / Bootcamp

Editing a Kelas Cohort / Bootcamp allows you to make changes to its details, ensuring that it remains up-to-date and accurate.

- Locate the Kelas Cohort / Bootcamp: In the Kelas Cohort / Bootcamp table, identify the Kelas Cohort / Bootcamp that you wish to edit.
- Access Edit Mode: Click on the associated icon, followed by "Edit." This action will open the editing interface for the selected Kelas Cohort / Bootcamp Link.
- Modify Payment Details: In the edit interface, you can make changes to various fields of the Kelas Cohort / Bootcamp. This includes updating the amount, expiry date, and more.
- Save Changes: After making the necessary edits, be sure to click the Save button to confirm and save your modifications.

### Viewing Payment Kelas Cohort / Bootcamp

You can access a comprehensive overview of each Kelas Cohort / Bootcamp by following these steps:

- Locate the Kelas Cohort / Bootcamp: In the table, identify the Kelas Cohort / Bootcamp and click on the associated icon, followed by View Details.
- Explore Insights: On the View Details page, you will discover a wealth of information, including payment link information, customer details, and payment history.
- Sharing: To share a Kelas Cohort / Bootcamp, click the "Share" button. You can copy the URL or directly share it on WhatsApp, Facebook, Twitter, Line & LinkedIn.
- QR Code Generation: Conveniently generate a QR code by clicking on the QR code icon. You can then copy it to the clipboard or download the image.

### Managing Kelas Cohort / Bootcamp Link

You get access to a default Kelas Cohort / Bootcamp in the Mayar dashboard under Kelas Cohort / Bootcamp. This default link, when shared to customers, will allow them to manually enter the amount they want to send/pay to you.

Below are the steps involved in managing the Default Kelas Cohort / Bootcamp

- Click on Kelas Cohort / Bootcamp in the left sidebar. You can copy the default link from here, which can then be directly shared with any customer. Additionally, you can add the link to any order-taking form or embed it as a button on your website.

### Locating QR Code, Deactivate Button, and View Details Page

- View QR Code: To access the QR code, you need to do checkout via VA or E-Wallet .
- Unpublished Default Link: If you wish to temporarily disable the default Kelas Cohort / Bootcamp and prevent further access, simply click dropdown published button, followed by selecting “unpublished”. This can be valuable when you need to suspend the link temporarily.
- View Details Page: For a deeper understanding of your default Kelas Cohort / Bootcamp, select "view" hyperlink at every each product. This action will navigate you to the Details Page, where you can explore revenue insights and access a complete payment history.
