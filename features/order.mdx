### Overview

This is the main page if you're selling physical product. You can download and print shipping label, request package pickup, and track the shipment in this page. Mayar's package pickup service and shipping is powered by KiriminAja.

### How to manage orders on Order Page?

Follow these simple steps to do an action Order Page in Mayar:

1. Choose Order on list.

<img
  src="/images/Screenshot2025-08-18152422.png"
  alt="Screenshot2025 08 18152422 Pn"
  style={{ width:"43%" }}
/>

2. Select "View" from the product hyperlink.

![Screenshot2025 08 18150845 Pn](/images/Screenshot2025-08-18150845.png)

3. Select Action to do on List at right side menu bar. 

   | Information      | Description                                                                                                                                                                                     |
   | :--------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | Packed           | Your order need to carefully packaged and is ready for shipping. You ensures that all items are securely packed to prevent any damage during transit, also you need to attach Label on package. |
   | Awaiting Courier | Your package is now waiting to be picked up by the courier. This means it will soon be on its way to you. We’ll update you once it’s on the move.                                               |
   | Shipped          | Your order has been handed over to the courier and is currently on its way to your specified address. You can track its journey using the tracking number provided.                             |
   | Completed        | Your package has been successfully delivered. We hope you’re satisfied with your purchase. If you have any feedback or need assistance, feel free to reach out to our customer service team.    |

![Screenshot2025 08 18152226 Pn](/images/Screenshot2025-08-18152226.png)

4. **Tracking:** Track detail status Order from Courier Pick up Order untill Courier done Deliver your Order.

   ![image.png](/images/image.png)
5. **Drop Off:** Drop your package at the nearest shipping service provider that available from your area. Make sure that package already attached Label & Order code.

   ![Screenshot2025 09 18132439 Pn](/images/Screenshot2025-09-18132439.png)


6. **Pick Up:** Request Pick Up your order by Courier from Shipping Service that available in your area. Make sure that package already attached Label & Order code.

   ![Screenshot2025 09 18132504 Pn](/images/Screenshot2025-09-18132504.png)
7. **Cancel Order:** Cancel your shipment and pickup order.

   ![Screenshot2025 09 18132523 Pn](/images/Screenshot2025-09-18132523.png)
8. **Label Product:** Print your physical product label and attach it to the package to be shipped.

   ![Screenshot2025 09 18132742 Pn](/images/Screenshot2025-09-18132742.png)

![Screenshot2025 09 18132956 Pn](/images/Screenshot2025-09-18132956.png)
